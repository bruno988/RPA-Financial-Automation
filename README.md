🚀 Automação de Processamento Financeiro
Projeto de RPA desenvolvido no Power Automate Desktop integrado com SQLite.

🛠️ Tecnologias Utilizadas
Power Automate Desktop: Lógica de automação e iteração.

SQLite (DB Browser): Armazenamento e persistência de dados.

SQL: Manipulação de dados (DML e DDL).

📋 Funcionalidades
Conexão direta com banco de dados local.

Filtragem de alunos com status "Pendente".

Conversão dinâmica de tipos de dados (String para Decimal).

Cálculo automatizado de descontos.
